● 3DU 프로젝트 분석 결과

  프로젝트 개요

  교육용 3D 물리 시뮬레이터 - 물리 법칙을 3D 시각화로 학습하는 웹 애플리케이션

  현재 구현된 모듈: 야구 투구 시뮬레이터 (중력, 항력, 마그누스 효과)

  ---
  기술 스택
  ┌───────────┬─────────────────────────────────────────────────┐
  │   분류    │                      기술                       │
  ├───────────┼─────────────────────────────────────────────────┤
  │ Frontend  │ React 18, TypeScript 5.2                        │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 3D 렌더링 │ Three.js, @react-three/fiber, @react-three/drei │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 상태관리  │ Zustand, React Context                          │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 스타일링  │ Styled-components                               │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 백엔드/DB │ Supabase (PostgreSQL, Auth, Realtime)           │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 빌드      │ Vite 5.0                                        │
  ├───────────┼─────────────────────────────────────────────────┤
  │ 테스트    │ Jest                                            │
  └───────────┴─────────────────────────────────────────────────┘
  ---
  주요 기능

  1. 물리 엔진
    - 4차 Runge-Kutta(RK4) 수치 적분
    - 중력, 항력, 마그누스 효과 계산
    - 릴리스 포인트 자동 보정
  2. 3D 시각화
    - 블렌더 투수 3D 모델 (FBX, 119프레임 애니메이션)
    - 실시간 궤적 렌더링
    - 힘 벡터 시각화 (중력/항력/마그누스)
    - 다중 카메라 프리셋 (포수/측면/투수 시점)
  3. 사용자 기능
    - 로그인/회원가입 (Supabase Auth)
    - 실험 저장/불러오기
    - 2개 실험 비교 모드
    - 리플레이 컨트롤
  4. 성능 최적화
    - 그래픽 설정 패널 (DPR, FPS, 안티앨리어싱 조절)
    - React.memo/useMemo 최적화

  ---
  프로젝트 구조

  src/
  ├── core/           # 물리엔진, 렌더러, UI 컴포넌트
  ├── scenarios/      # 시나리오별 구현 (pitch/)
  ├── components/     # 공통 UI (Button, Card, Input)
  ├── contexts/       # 전역 상태 (Simulation, Graphics, Comparison)
  ├── pages/          # 로그인/회원가입 페이지
  ├── lib/            # Supabase 설정
  └── utils/          # 유틸리티 함수

  ---
  교육적 효과

  - 포물선 운동, 마그누스 효과 직관적 이해
  - 회전에 따른 구종 변화 시각화 (포심, 커브, 슬라이더 등)
  - 힘 벡터 실시간 표시로 물리 법칙 학습
